This is basically an interactive timeline. It is fully usable. You can add data and decide how it shows. You can define relations and add lines and arrows between related events.
The data can be gathered from Wikidata, to speed up the process and to make translation easy. 
Your project won't run on a server. You save it by exporting it as a JSON-file and you can import it back, whenever you want. 
The primary use, as I see it, is for study and schools.
